🔑 How to Get Your API Key

AgentForge supports multiple AI providers. Pick one below to get started — Gemini has a free tier.

🟢 Google Gemini FREE TIER

Gemini 2.5 Flash is free and excellent for most tasks. Recommended for beginners.

1. Go to aistudio.google.com/apikey
2. Sign in with Google account
3. Click Create API Key
4. Copy key starting with AIza...

Format: AIzaSy...xxxx

🟣 Anthropic Claude PAID

Claude Sonnet/Haiku — powerful and great for code. Requires billing setup ($5 min).

1. Go to console.anthropic.com
2. Create account → Settings → API Keys
3. Click Create Key
4. Copy key starting with sk-ant-...

Format: sk-ant-api03-...xxxx

🔵 OpenAI GPT PAID

GPT-4o Mini is affordable. Requires credit card ($5 min).

1. Go to platform.openai.com/api-keys
2. Create account → Billing → Add payment
3. API Keys → Create new secret key

Format: sk-proj-...xxxx

⚫ xAI Grok PAID

Fast and capable. Go to console.x.ai → API Keys.

Format: xai-...xxxx

🔷 DeepSeek PAID — Very Cheap

Extremely affordable (~$0.001/1K tokens). Go to platform.deepseek.com → API Keys.

Format: sk-...xxxx
💡 Tip: After entering your key in the header bar, click Save then Test to verify it works before analyzing a project.
